# HNL-Score-Predictor
Web aplikacija koju bi korisnici mogli koristiti za predviđanje rezultata u Hrvatskoj nogometnoj ligi te na temelju predikcija ostvarivati bodove. U repozitoriju se nalazi kod za Frontend (React JS) i Backend (Spring Boot - Java) te je za lokalno pokretanje potrebno povezati vlastitu bazu podataka jer web aplikacija nije trenutno deployana na internet.
